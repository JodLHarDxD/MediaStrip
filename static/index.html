— TECH STACK

Powered by the best.
Open-source backbone.

⚡

FastAPI

Backend Framework

Async Python web framework with automatic OpenAPI docs, SSE streaming, and sub-millisecond routing. Handles concurrent downloads without blocking.

📥

yt-dlp

Media Extraction

Fork of youtube-dl with active maintenance. Supports 1000+ sites, live progress streaming, and adaptive format selection for best quality.

👁️

OpenCV

Computer Vision

Frame sampling, temporal variance analysis, and watermark region detection. TELEA inpainting as CPU fallback when GPU isn't available.

🧠

LaMa

AI Inpainting

Large Mask Inpainting — a state-of-the-art neural network that reconstructs missing image regions using Fourier convolutions and perceptual loss.

🎬

ffmpeg

Media Processing

Stream-copy downloads bypass re-encoding entirely. For watermark output, preserves audio track and muxes clean video with original sound.

🔥

CUDA

GPU Acceleration

LaMa inpainting runs on NVIDIA GPUs via PyTorch + CUDA. 1080p video watermark removal completes in minutes instead of hours.
